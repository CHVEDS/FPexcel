# Excel Parser

Парсер Excel-файлов с вычислением формул, извлечением изображений и метаданных ячеек.

**Автор:** Чураев Вадим Эдуардович || РЭУ ИМ.ПЛЕХАНОВА


---

## Описание

Парсер предназначен для чтения сложных `.xlsx`-таблиц:
- вычисляет результаты формул (включая русские функции: `ЕСЛИ`, `ВПР`, `ЕСЛИОШИБКА` и др.) через Microsoft Excel
- извлекает текст формул из ячеек
- сохраняет изображения, встроенные в ячейки
- определяет объединённые ячейки и их принадлежность
- собирает метаданные форматирования (цвет заливки, жирность, шрифт)
- автоматически определяет реальные границы данных (игнорирует пустые колонки-«мусор»)

Разработан для работы с таблицами отслеживания закупок и контрактов, содержащими многострочные заголовки, цветовые группы, цепочки `IF/AND/IFERROR/VLOOKUP` с внешними ссылками.

## Быстрый старт

### Запуск из командной строки

```bash
python "parser (1).py" <путь_к_файлу.xlsx> [папка_для_изображений] [имя_листа]
```

Примеры:

```bash
# Активный лист, изображения в папку images/
python "parser (1).py" "Практика Python.xlsx"

# Указать лист и папку
python "parser (1).py" "Практика Python.xlsx" output_images "Исполнение"
```

### Использование как модуль

```python
from parser_module import parse_excel, extract_headers

# Базовый вызов
data = parse_excel("Практика Python.xlsx")

# С формулами и форматированием
data = parse_excel(
    "Практика Python.xlsx",
    image_dir="images",
    sheet_name="Исполнение",
    include_formula=True,       # возвращать текст формул
    include_formatting=True,    # возвращать цвет/шрифт ячеек
    max_col=66,                 # ограничить колонки (для файлов с мусорными столбцами)
    skip_empty=True,            # пропускать пустые ячейки
)

# Получить заголовки (строки 2 и 3 — как в оригинале)
headers = extract_headers(data, header_rows=[2, 3])
# {"B": "№ п/п", "C": "Код позиции (группы) КС", ...}

# Обход ячеек
for coord, info in data.items():
    print(coord, info["value"], info.get("formula"), info.get("image_path"))
```

## Возвращаемые данные

Словарь с ключом = координата ячейки (`"A1"`, `"BN23"` и т.д.):

| Поле | Тип | Описание |
|---|---|---|
| `value` | `str / int / float / datetime / None` | Вычисленное значение (через Excel) |
| `formula` | `str / None` | Текст формулы, например `=IF(C4="","",1)` |
| `image_path` | `str / None` | Путь к сохранённому PNG-изображению |
| `merged_into` | `str / None` | Координата верхнего левого угла объединённой области |
| `formatting` | `dict` | `bold`, `font_name`, `font_size`, `fill_color`, `number_format` |

---

## Параметры `parse_excel()`

| Параметр | По умолчанию | Описание |
|---|---|---|
| `filepath` | — | Путь к `.xlsx` файлу |
| `image_dir` | `"images"` | Папка для сохранения изображений |
| `sheet_name` | `None` | Имя листа; `None` = активный лист |
| `close_app` | `True` | Закрыть Excel после чтения |
| `skip_empty` | `True` | Пропускать ячейки без значений, формул и изображений |
| `include_formula` | `True` | Включать текст формул в результат |
| `include_formatting` | `False` | Включать данные форматирования |
| `max_col` | `None` | Максимальная колонка (авто-определение до 500) |
| `max_row` | `None` | Максимальная строка (без ограничения) |

## Тестовый файл `test_data.xlsx`

Файл воспроизводит реальную структуру таблицы отслеживания закупок:

**Лист «Исполнение»** — 20 строк данных, колонки B–BN:
- Строки 2–3: двухуровневые заголовки с объединёнными ячейками и цветовыми группами
- Цветовые группы: Балансовая стоимость (жёлтый), АКТ/Факт (синий), Связка с ЦФУ (розовый), Связка ОФЗ (зелёный)
- Формулы: `=IF(C4="","",1)`, `=IFERROR(VLOOKUP(...))`, `=IF(AND(...),Y4,"")` и др.
- Три типа строк: завершённые (AR=1), закрытые без факта (J заполнен), активные в работе (G=1)
- Встроенное изображение в ячейке H4

**Лист «Справочник»** — 12 записей-контрагентов (аналог внешнего `[1]Нормативная документация`)

Пересоздать тестовый файл:

```bash
python create_test_data.py
```

## Известные ограничения

- Требует установленного **Microsoft Excel** (xlwings запускает Excel в фоне для вычисления формул)
- Формулы с ссылками на **закрытые внешние файлы** могут вернуть `None` или ошибку — Excel попытается открыть внешний файл
- Файлы со стилями на тысячах пустых колонок (например, A1:XCA20323) автоматически обрезаются до реального диапазона данных (до 500 колонок)

# Excel Parser: Формулы + Изображения (с поддержкой Excel)

Парсер Excel-файлов на Python, который корректно обрабатывает **сложные формулы** (включая русские функции и внешние ссылки) и **извлекает изображения**, привязанные к ячейкам.

Автор: **Чураев Вадим Эдуардович**

---

## ✅ Возможности

- Поддержка формул на **русском языке**: `ЕСЛИ`, `ВПР`, `ЕСЛИОШИБКА`, `ИСТИНА`, `ЛОЖЬ` и др.
- Обработка **внешних ссылок**: `'C:\...\[файл.xlsx]Лист'!A1:B10`
- Извлечение **изображений**, вставленных в ячейки
- Возврат **вычисленных значений** (как их видит пользователь в Excel)
- Поддержка всех типов данных: текст, числа, даты, ошибки (`#N/A`, `#REF!` и т.д.)

---

## ⚠️ Требования

- **Операционная система**: Windows или macOS
- **Установленный Microsoft Excel** (2010 или новее)
- Python 3.8+

> ❌ Не работает на Linux без эмуляции Excel (например, через Wine + COM).
